# Utilitários de Saúde 🏥
Um programa de terminal com ferramentas práticas para calcular indicadores de saúde, desenvolvido em Python.

> Status do Projeto: Em desenvolvimento ⌛

## Funcionalidades

- **Calculadora de IMC** - Calcula o Índice de Massa Corporal e indica a categoria (abaixo do peso, normal, sobrepeso, etc.)
- **Calculadora de TMB** - Calcula a Taxa Metabólica Basal, ou seja, quantas calorias seu corpo precisa por dia em repouso
- **Calculadora de Hidratação** - Indica quantos litros de água você deve beber por dia com base no seu peso
- **Conversor de Unidades** - Converte kg ↔ libras, cm → polegadas e temperaturas entre °C e °F


## Como executar

Você precisa ter o [Python](https://www.python.org/downloads/) instalado na sua máquina.

Clone o repositório:
```bash
git clone https://github.com/seu-usuario/utilitarios-saude.git
cd utilitarios-saude
```

Execute o programa:
```bash
python saude.py
```

## Exemplo de uso

```
========================================
BEM-VINDO AOS UTILITÁRIOS DE SAÚDE
========================================
1 - Calcular IMC
2 - Calcular TMB (calorias diárias)
3 - Calcular hidratação diária
4 - Converter unidades de saúde
0 - Sair
========================================

```
